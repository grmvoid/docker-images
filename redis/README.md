## How to use this image

### start a redis instance

```console
$ docker run --name redis -d sxbrsky/redis:latest
```

### ... via [`docker-compose`](https://github.com/docker/compose)
Example `docker-compose.yml` for `redis`:

```yaml
version: '3.1'

services:
    redis:
        image: sxbrsky/redis:latest
        restart: always
        ports:
            - "6379:6379"
```
